#  Repositorio de Prácticas en Java

##  Descripción General

Este repositorio contiene tres programas desarrollados en **Java**, donde se aplican estructuras de control, arrays, entrada de datos por teclado y generación de números aleatorios.

Los ejercicios fueron realizados como parte del módulo de *Entornos de Desarrollo*.


#  Programas incluidos

##  1️ Eliminación de un número en un array (Main1.java)

Este programa:

- Solicita al usuario la cantidad de números a introducir.
- Guarda los números en un array.
- Pide un número específico para eliminar.
- Genera un nuevo array sin ese número.
- Muestra el resultado final.

###  Conceptos aplicados:

- Arrays
- Bucles `for`
- Contadores
- Uso de `Scanner`
- Comparación de valores
- Creación de un nuevo array dinámico

> “Un array permite almacenar múltiples valores del mismo tipo en una sola estructura de datos.”


##  2️ Verificación de mayoría de edad (Main2.java)

Este programa:

1. Pide al usuario varias edades.
2. Almacena las edades en un array.
3. Verifica si todas las personas son mayores de edad.
4. Devuelve un valor booleano (`true` o `false`).

###  Conceptos utilizados:

- Arrays
- Variable booleana
- Estructura condicional `if`
- Uso de `break`
- Bucle mejorado `for-each`

> *El uso de variables booleanas permite simplificar validaciones lógicas.*


##  3️ Búsqueda de número en array aleatorio (Main3.java)

Este programa:

- Genera automáticamente 10 números aleatorios entre 1 y 100.
- Solicita un número al usuario.
- Busca si el número existe en el array.
- Indica la posición donde se encuentra.

###  Contenidos aplicados:

- Clase `Random`
- Clase `Scanner`
- Arrays de tamaño fijo
- Búsqueda secuencial
- Uso de bandera booleana (*flag*)
- Estructura condicional
